Javascript

Linguagem criada em 1995 e que é executada diretamente no browser. Ela permite que você inclua interatividade em sua página. Se você quiser testar o javascript você pode acessar o developer tools no chrome (ctrl + shift + I) e acessar o console.

Tipos de dados

  • Numbers
  • Boolean
  • String: "John", "a", ""
  • Object: existem dois tipos -> Dictionary: { "first_name" : "John" , "last_name" : "Lennon"}, Array: ["john", "paul", "ringo", "george"].
  • null: ausência de valor
  • undefined: ainda não foi definido, usado para variáveis.

Comandos e operadores

  • if (condição) {comando(s);} else {comando(s);}
  • "==" usado para comparar valores, "===" usado para comparar valor e tipo.
  • "&&" representa and
  • "||" representa or
  • "!" representa not
  • "!==" representa não iguais

Loops

  • for (var i = 0; i < students.length; i += 1){comando(s);}
  • while (condição){comando(s);}

Functions

function nameOfTheFunction(parâmetros,...){ comando(s); return "alguma coisa";}

DOM

Estrutura de um arquivo html, que basicamente é uma estrutura em árvore. Sendo que a tag html é a raiz.
